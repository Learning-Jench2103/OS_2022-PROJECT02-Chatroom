# 110-1 Operating System - Project 2

## Usage
- Compile from source code
    ```shell
    $ make
    ```
- Execute the server program
    ```shell
    $ ./bin/server <IP_ADDRESS>:<PORT>
    ```
- Execute the client program
    ```shell
    $ ./bin/client <IP_ADDRESS>:<PORT>
    ```
- Remove all the executable files
    ```shell
    $ make clean
    ```
